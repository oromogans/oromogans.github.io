ChatGPT for learning new languages

I saw a video on YouTube that changed how I think about learning new languages. It made so much sense that I decided to try this technique next time I learn a new language. But there’s a big problem. If you want to try this method, you need a native speaker of the language you want to learn. Even if you find one, you still have to plan the lessons and talk about payment.

That’s where ChatGPT comes in and it’s a real game-changer. I can ask it to create short stories with translations of difficult words. I’m now starting to learn Serbian, and I use ChatGPT to write a story and ask some questions about it, then correct my answers.

Me
Today you are the best teacher of Serbian Language. 
To learn serbian we will use storytelling technique. 
This plan: You write a little story, then ask 2-3 questions. 
I write answers and then you write my mistakes and how to avoid them.
Ok? Let's start with simplest sentences

ChatGPT

Sounds like a great plan! Let's get started.

**Story:**

Jednom davno, živela je devojčica po imenu Mila. 
Mila je bila srećna devojčica koja je volela da trči po parku 
i igra se sa svojim prijateljima. Svaki dan, ona bi išla u park
sa svojim psom, koji se zvao Žuća. Mila i Žuća su uvek pravili puno zabave.

**Questions:**

1.  Kako se zvala devojčica?
2.  Šta je Mila volela da radi?
3.  Kako se zvao njen pas?

Now, you can answer these questions in Serbian.

I want to combine this approach with Anki flashcards and see how quickly I can learn a new language.
